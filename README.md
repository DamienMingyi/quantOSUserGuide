# 关于quantOS

quantOS是一个量化开源社区，提供量化开源系统的一站式解决方案。

在这里，我们希望与您一起：

* 开发完备的量化投资工具
* 探索科学的量化研究方法
* 交流一切关于量化投资的有意思的事情

社区的创始会员有《乱世华尔街》的作者，有tushare和vn.py的创始人，还有一大波技术大牛。

详细的信息，可以登录[官方网站](http://www.quantos.org)了解。
